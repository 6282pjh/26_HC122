# 월드 모델 기반 행동 시뮬레이션을 제공하는 수술 어시스턴트 로봇

## 1. 프로젝트 소개
본 프로젝트는 수술 과정에서 반복적으로 수행되는 도구 전달, 절개 보조, 봉합 보조 작업을 집도의와 협업하여 수행하는 **Human-in-the-loop 기반 협업형 수술 보조 로봇 시스템**을 개발하는 것을 목표로 합니다.

기존 수술 로봇이 정밀한 원격 조작에 강점을 가졌다면, 본 시스템은 한 단계 더 나아가 **로봇의 다음 행동을 실제 수행 전에 먼저 예측·시뮬레이션하고, 집도의의 승인 및 수정 이후에만 실행**하는 구조를 제안합니다.

핵심은 NVIDIA Cosmos 기반 월드 모델과 디지털 트윈 환경을 활용해 절개, 봉합, 도구 전달과 같은 보조 동작을 가상환경에서 먼저 검토하고, 그 결과를 집도의에게 직관적으로 제시하는 것입니다. 집도의는 절개 길이, 깊이, 위치, 방향, 도구 전달 순서, 봉합 범위 등을 직접 수정하거나 승인할 수 있으며, 로봇은 이 피드백을 반영해 최종 행동 계획을 재구성하고 수행합니다.

---

## 2. 프로젝트 개요
- **프로젝트명**: 월드 모델 기반 행동 시뮬레이션을 제공하는 수술 어시스턴트 로봇
- **진행 기간**: 2026.04.01 ~ 2026.10.30
- **주제 영역**: 생활 / 업무 / 의료
- **기술 분야**: 소프트웨어 / 인공지능(AI) / 스마트디바이스
- **성과 목표**: 논문 게재 / 앱 등록 / 실용화 / 공모전 참가

---

## 3. 제안 배경
현대의 수술 환경은 환자별 해부학적 구조와 조직 상태가 다르고, 작은 오차가 결과에 큰 영향을 줄 만큼 높은 정밀도와 실시간 판단 능력을 요구합니다.

또한 의료 현장에서는 의료 인력 부족과 반복적인 보조 업무 부담이 누적되고 있으며, 수술 과정에서 의료진의 피로도와 업무 효율 저하가 중요한 문제로 대두되고 있습니다.

이러한 환경에서 로봇이 모든 판단을 대신하는 완전 자율형 구조보다, **집도의가 최종 판단권을 유지한 채 로봇이 정밀 작업과 반복 작업을 보조하는 협업형 구조**가 현실적이고 안전합니다.

따라서 본 프로젝트는 수술 보조 로봇을 단순 자동화 장치가 아니라, 행동을 먼저 예측하고 사람이 검토·수정한 뒤 실행하는 **안전 중심 협업 시스템**으로 정의합니다.

## 4. 주요 기능
### 4.1 수술 환경 및 도구 인식
- 3D 비전 카메라를 통해 수술 공간과 도구의 위치를 인식
- 정밀 작업에 필요한 좌표 및 환경 정보 확보

### 4.2 월드 모델 기반 미래 행동 예측 시뮬레이션
- 절개, 봉합, 도구 전달 동작을 실제 수행 전에 가상환경에서 시뮬레이션
- 동작의 적절성과 안전성을 사전에 검토 가능

### 4.3 피드백 기반 행동 수정
- 집도의가 절개 깊이, 위치, 방향, 도구 전달 순서, 봉합 범위 등을 수정
- 로봇이 이를 반영하여 행동 계획을 재조정

### 4.4 정밀 수술 보조 수행
- 승인된 계획에 한해 mm 단위 절개 보조, 도구 전달, 봉합 보조 수행

### 4.5 디지털 트윈 기반 검증 시스템
- 실제 로봇과 연동된 가상환경에서 동작을 먼저 검증
- 안전성과 신뢰성을 높이고 향후 Sim-to-Real 기반 고도화 가능
